Instructions

  • Clean up your website. If there is code that isn't being used, remove it. Make it so the code in VS Code looks nice.

  • Go through the list in the Project Overview Page that says: EVERY PAGE SHOULD and ensure that each page is good to go. The biggest aspect of this is likely the media queries to ensure that the pages look good on all devices.

  • Go through the list in the Project Overview Page that says: THE PROJECT SHOULD and make sure that the project is good to go. The only thing here you'll likely have to worry about is publishing your website. The Lesson 5 Reading shows exactly how to publish your website.

  • Finally, submit your project. This time, all you'll have to do is submit the url to your live website. You can submit your project Here. Be sure to submit this whether it is finished or not by the end of our Lesson 6 Class Session.
