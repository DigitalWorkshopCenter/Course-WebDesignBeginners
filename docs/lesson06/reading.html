There are many ways to publish a website. When we say publish, we mean to actually put it onto the web so that anyone could access it with a url.

The ways that we publish websites gradually change and adapt with time. Thankfully, it is continually getting easier and easier to publish websites. Some of the easiest methods (that use version control like git), we will not use in this course because there is a bit of learning that needs to take place before those can be used.

However, there are still easy methods to deploy a website. Several of these include:

  • 000webhost

  • Fast.io

  • Netlify

The easiest of these by far is Netlify, and Netlify is actually another really good option even when you start incorporating version control such as git, which we will do in future classes.
