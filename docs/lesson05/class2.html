Hands-on Exercise 2
Links, Links, and More Links

Overview

In this activity, you will write your own links, image links, email links, div links and entire page links.

Instructions

  1. Create a new html file.

  2. Open this file with Live Server

  3. Create an a tag to create a simple link to navigate to youtube.com. Test it in your browser to ensure that the link works.

  4. Create a new link with an image from google images. This link should navigate to a website that has to do with the image you selected. Again, test this in the browser and make any necessary adjustments until it is working as intended.

  5. Create a new link that will open up the default email program used by a user. This link should include a recipient email, a subject and a body so that when the user clicks on it, there is almost a completed email there ready to send. Test this to ensure that it works.

  6. Create a div that is 300px tall and 300px wide with a bright background color. Put some text into this div so that it shows up on the page. Now, put an a tag around the div so that the entire div becomes a link to google.com. The link should open up a new tab.

  7. Create an a tag that spans the entire body of the page. This link can go anywhere you want it to. Test it to make sure it works.

Conclusion

Congratulations! You are now a link master!
