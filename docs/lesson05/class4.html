Hands-on Exercise 4
CSS Transitions

Overview

In this activity, you will create your first CSS Transition.

Instructions

  1. Create a new html file and open it with Live Server

  2. Create an image on the page, make it look nice but only set it to be 400px wide.

  3. Add a transition rule to the image css with a duration of your choosing.

  4. Add a hover rule to this image to make the image grow to be 600px wide when the user hovers their mouse over the image.

  5. This article may be a helpful resource in helping you accomplish this task.

Conclusion

Congratulations! You now know how to create simple css transitions!
