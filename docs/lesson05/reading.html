According to W3 Web Accessibility is when websites and web tools are properly designed and coded, people with disabilities can use them. However, currently many sites and tools are developed with accessibility barriers that make them difficult or impossible for some people to use.

This Stardust Article had several good suggestions for increasing web accessibility. They include:

  • Use a CMS that supports accessibility

  • Structure and organize with headers

  • Assign alt tags for images

  • Use descriptive title for links

  • Use colors carefully

  • Design forms for web accessibility

  • Use tables for tabular data

  • Ensure site navigation via a keyboard

  • Turn dynamic content into accessible content

  • Validate web accessibility
