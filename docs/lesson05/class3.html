Hands-on Exercise 3
CSS Gradients

Overview

In this activity, you will create your first CSS Gradients.

Instructions

  1. Create a new html file and open it with Live Server

  2. Modify the body style of this html page by giving it a background gradient...any of your choosing.

  3. There are a number of resources to help you learn how to do this, but as I've said in class before, I really enjoy using css gradient generators , then making further customizations from there. This is also a really great way to see how the different css rules change the behavior of the gradients that are being used.

  4. Once that body background is working the way you want it to, create a div on the page that takes up exactly 50% width of the page and 100% height of the page. Create a new gradient background for this div.

Conclusion

Congratulations! You now know how to create css gradients!
