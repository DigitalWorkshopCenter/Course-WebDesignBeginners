Assignment #5

Overview

For your assignment, you will gain experience working with advanced CSS.

Instructions

The goal of this exercise is to use what you learned during the lesson in your own way.

  1. Create a new html file.

  2. Create a new css file and link it to the html file.

  3. Create a button and for the background color, use a linear gradient at a 45 degree angle.

  4. Using a transition, make it so the button looks different when the user hovers over it.

  5. Create a div on the page with a radial gradient.

  6. Using a CSS Animation, make it so the div with the radial gradient has a loop that has it change colors forever.

  7. Create an html form of your choosing and make it look nice.

Stretch Challenges

  • Add some other awesome CSS thing that you want to try.

  • Use JavaScript to make a popup window when the form submit button is pressed...you'll probably have to do some Google Searching to make this happen.
