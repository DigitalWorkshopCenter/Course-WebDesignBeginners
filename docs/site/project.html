Lab Project Overview
One Awesome Website for One Awesome Person

Project Overview

This course will have 3 labs (lessons 2, 4 and 6). Each lab will build on the previous lab, applying the concepts learned in other class sessions. By the end of the third lab session (lesson 6), you will finish and publish an awesome website about yourself to the web.

You should invest your time into this project to make it look good and work well. In the world of web development, the more you can market yourself and your skills, the better jobs you will be able to get. The more time you put into it, and the more questions you ask, the further along with HTML and CSS you will be â€” and the more equipped you will be with future projects and employment.

The project that you are going to build is a website about you. It's purpose is to give you an opportunity to practice writing HTML and CSS, and then have something fun to show people, and something professional to show potential employers.

Project Requirements

This website should include the following components:

  • A professional homepage

    • Must be named index.html and

    • Include a picture of you, your family, or something that represents you.

    • Includes a description about yourself.

    • Includes links to the other sections of your website within the homepage (not just the navbar)

  • Favorite Stuff Page

    • This can be either a Favorite Places or Favorite Foods page.

    • Include at least 5 favorites. Each should have a title, image and description.

  • Fun Stuff Page

    • This can be either a Fun Experiences or Funny Stories page.

    • Include at least 5 favorites. Each should have a title, an approximate date, 1 image or 1 video and a description.

  • Good Stuff Page

    • This can be either a Hobbies or Skills page.

    • Include at least 5 items. Each should have a title, an image or video, and an HTML list showing either how you came to have that skill or hobby or things you have done with that hobby or skill.

  • Personal Stuff Page

    • This should be a webpage of your choosing. It should fit well with the other pages in being about you.

    • You may include whatever you want to on this page, but it needs to include at least 3 images, and an HTML table.

  • EVERY PAGE SHOULD...

    • ...be in its own folder that includes the relevant css file and any relevant images or other media content.

    • ...have a css file for that page.

    • ...have a professional layout on desktop, tablet and mobile screens by using CSS media queries

    • ...have a Navbar that looks the same (except an active identifier to show the user which page they're on).

    • ...have valid links and valid html.

  • THE PROJECT SHOULD...

    • ...have a professional architecture and folder structure.

    • ...be published to the web.

Stretch Challenges

  • Include individual pages for both Favorite Places and Favorite Foods

  • Include individual pages for both Fun Experiences and Funny Stories

  • Include individual pages for both Hobbies and Skills

  • Include a unique HTML element on each page (for example, media, iframe, table, list, etc.)
