Hands-on Exercise 5
Working with Links

Overview

This is a simple exercise that is intended to give you practice working with links.

Instructions

  • Download the Starting Zip Folder

  • Unzip it, and open the folder in VS Code

  • Open the index.html file with Live Server

  • Look at the comments in the HTML of each file. They show the instructions of what you need to add, and where you need to add it in order to complete this activity.
