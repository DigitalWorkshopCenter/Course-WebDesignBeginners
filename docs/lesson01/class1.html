Hands-on Exercise 1
Creating Modules

Overview

The focus of this assignment is to create our first JavaScript Modules. Here are our goals for today:

  1. Create an HTML file to run in the browser that links to a JS file

  2. Create a JS file for the html file

  3. Create a second JS file that is a module, that will be referenced in the first JS file

Instructions

Code without Modules

  1. Create an html file named app.html

    • Include an input for a user to enter the radius of a circle

    • Include a button that will call a function when clicked

    • Include a div with an id of "output" for the circumference to be displayed from the JS function

  2. Create a js file named app.js

    • Create a function called submitFunction()

    • Use document.getElementByID to retrieve the value from the html input

    • Use parseFloat() to convert the input string to a number

    • Use the formula Ï€ r^2 to compute the circumference of a circle

    • Use document.getElementByID to display that circumference in the output div.

  3. Add a script tag to the html file that references app.js

  4. Write the code or use onclick in the button to call submitFunction() when the button is clicked.

  5. Run the code and ensure that everthing works as intended.

Code With Modules

Now that you have the code working the way that we're used to, let's split up our logic into modules.

  1. Create a file called circleCalculator.js. In it:

    • Create a function called calculateCircumference(r)

    • Export this function by inserting the text export before function where it is declared.

  2. In app.js import the calculateCircumference function at the top of the file in this manner: 

    
    import {calculateCircumference} from './circleCalculator.js';

  3. In submitFunction() call the calculateCircumference function using the value from the user input as the argument to be passed into the function.

  4. Lastly, because of scope being different with modules, add code to trigger the onclick function call from JavaScript instead of from the HTML:

    • In the html, add an id attribute to the button with a value of "submitButton"

    • In app.js, add the following code to trigger the function when a click is detected: 

      
    document.querySelector('#submitButton').onclick = submitFunction;
