Hands-on Exercise 4
Our First Callback Functions

Overview

This is a very simple exercise that is intended to give you practice writing and running callback functions.

Instructions

  • Go through this article until you've completed the Callback as an Arrow Function section. With each piece of code you see in it, write/copy that code into your own file and run it and see what happens.

  • After you have completed each piece of code in the article above, think about your own callback function and write it. It can be anything and does not have to be extremely useful. This is just intended to give you practice working with callback functions.
