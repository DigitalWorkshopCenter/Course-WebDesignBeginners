Hands-on Exercise 4
Working with Lists

Overview

This is a very simple exercise that is intended to give you practice working with HTML lists.

Instructions

  • For your first list, you will create a numbered list, with your 5 favorite cities in order from most favorite to least favorite.

  • Create an unordered list with 10 fruits or vegetables that you can think of.

  • Create a list with the names of the people in this class

  • Create one more list of your choosing.

  • Now, try to add style to these lists, and make each one look a little bit different. It will probably be easiest to do this using classes.
