Creating More Modules

Overview

This exercise will start where we left off with the first exercise today. The zip file for that code can be found here.

In this exercise, we will create an additional module, as well as additional functions to help make our circle program more useful for our users.

Instructions

  1. Modify app.html by adding another label, input, button and output div for the area of a circle.

  2. Modify circleCalculator.js by giving it a function called calculateArea()

  3. Create a new js file called validation.js

    • Create a function called convertToNumber() that will accept a parameter and return either an error if it couldn't covert it to a number, or a the data converted to a number.

  4. Modify app.js by including the import to convertToNumber() as well as calling that function in both cases where we receive input from the user.
