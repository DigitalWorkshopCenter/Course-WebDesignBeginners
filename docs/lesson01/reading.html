How to Create a Website

Websites are built using code. Generally, websites consist of multiple types of code, or programming languages. In this course, we will learn how to build web pages and make them look awesome using HTML and CSS.

To understand how to write code in a specific language, you need a couple of things. First, you need a tool to write it. Second, you need a place to execute it. For example, if I want to record a video, I could use Zoom. Then, once I have that video file/recording, I need a program to watch/open/execute it like VLC Media Player. When we write HTML and CSS code, we generally write it in a text editor. You can think of a text editor as being very similar to Microsoft word. If I want to write an English paper, I will use Word. If I want to write a webpage consisting of HTML and CSS, I will use a text editor.

This is where VS Code comes in. VS Code is an incredible text editor that is used widely in the workplace and that we will use in this course.
