Assignment #1

Overview

Today, you will complete your first assignment for this course. You will have an opportunity to demonstrate your understanding of the concepts taught in this module, and surely you will learn a lot as you develop a website of your own. These assignments are designed to be both fun and educational, as well as give you an opportunity to stretch yourself and develop your skills.

Instructions

Your tasks are as follows:

  • Think of a website you would like to create.

  • Create a file called index.html which will serve as your home page.

      Format index.html so that it includes a header and footer.

    • Between the header and footer, include three divs. Inside these divs could be headers, paragraphs, images...whatever you need for the home page of your site.

    • Incorporate styles as classes so that your html looks professional.

  • Create another html page that your website will need.

    • Include at least 5 html elements in the body of this page.

    • Incorporate styles as classes so that your html looks professional.

  • Be sure to include links in the pages so that you can navigate between the two of them.

  • Include a list somewhere on one of these html pages.

Stretch Challenges

  • Include a third html page.

  • Include images or other media.

  • Move all of your css to a separate css file.
