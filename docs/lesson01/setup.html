VS Code & Chrome Setup

VS Code

To set up our computers to write JavaScript, all we need to do is download a text editor. We will be using Visual Studio Code also known as VS Code.

  1. Open this webpage in your browser.

  2. Download VS Code for your operating system.

  3. Install VS Code.

  4. Accept all the default settings.

Now that it is installed. Open VS Code. We are going to download two extensions. Click on the Extensions Icon and search for Live Server and click install.

Lastly, search for Prettier - Code Formatter and install it. Watch this video to see how to configure it.

Additionally, you may also watch the following media content that includes helpful tips and tricks to help you work efficiently with this program.

  • Visual Studio Code Tutorial for Beginners - Introduction (By Academind)
