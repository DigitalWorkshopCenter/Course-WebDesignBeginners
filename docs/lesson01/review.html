We declare variables in JavaScript using the let keyword. JavaScript includes the following data Structures (datatypes):


    let name;                                                // undefined
    let todayIsRainy = false;                                // boolean
    let studentAge = 20;                                     // number
    let myName = "Nathan Birch";                             // string
    let studentNames = ["Nancy", "Fouzi", "Byron", "Gabe"];  // array
    let person = {name: "Jake", age: 25, isHappy: true};     // object
    let function = computeArea(width, height) {              // function
        return width * height 
    };

JavaScript supports the following arithmetic operators


    let x = 5 + 2; // addition       (x is 7)
    let y = x - 3; // subtraction    (y is 4)
    let z = x * y; // multiplication (z is 28)
    y = y / 2;     // division       (y is 2)
    x = x ** 2;    // exponentation  (x is 49)
    z = x % 2;     // modulus        (z is 1)
    z++;           // increment      (z is 2)
    z--;           // decrement      (z is 1)

...and the following arithmetic shortcuts


    let x = 5; // x is 5
    x += 2;    // x is 7
    x -= 1;    // x is 1
    x *= 6;    // x is 6
    x /= 2     // x is 3
    x %= 2     // x is 1

JavaScript supports the following Conditional Operators and when used in an expression, they form a boolean expression, meaning their output is a true/false value as shown below


    let x = 5, y = 10; // declare variables for the following examples
    x == y             // is equal to value:              false
    x === y            // is equal to value and type:     false
    x != y             // is not equal to value:          true
    x !== y            // is not equal to value and type: true
    x > y              // is greater than:                false
    x >= y             // is greater than or equal to:    false
    x < y              // is lesser than:                 true
    x <= y             // is lesser than or equal to:     true

JavaScript supports the following Logical Operators


    (5 < 6 && 7 === 8)    // AND: false
    (5 < 6 || 7 === 8)    // OR: true
    (5 < 6 && !(7 === 8)) // NOT: true

If statements in JavaScript look like the following: 


    if (day === "Sunday" && time === "9:00am") {
        alert("Go to Church");
    } else if ((day === "Monday"    || 
                day === "Tuesday"   ||
                day === "Wednesday" ||
                day === "Thursday"  ||
                day === "Friday") && time === "8:00am") {
        alert("Go to School");
    } else {
        alert("Go Play!");
    }

JavaScript implements the following loop types: 


    // for loop example            
    for (let step = 0; step < 5; step++) { 
        // Runs 5 times, with values of step 0 through 4.
        console.log('Walking east one step');
    }

    // while loop example
    let n = 0;      
    let x = 0;
    while (n < 3) {
        n++;
        x += n;
    }

    // do while loop example    
    let i = 0;    
    do {
        i += 1;
        console.log(i);
    } while (i < 5);

    // for in loop example    
    for (let i in Math) {     
        console.log(i + ' = ' + Math[i];
    }

    const arr = [3, 5, 7];
    arr.foo = 'hello';
    // for in example    
    for (let i in arr) {       
        console.log(i); // logs "0", "1", "2", "foo"
    }

    // for of example    
    for (let i of arr) {       
        console.log(i); // logs 3, 5, 7
    }

JavaScript can manipulate the DOM in many ways. The following code should be familiar to you.


    // run this function when the document is loaded
    window.onload = function() {
        // create an HTML header
        const heading = document.createElement("h1");
        const heading_text = document.createTextNode("Big Head!");
        heading.appendChild(heading_text);
        document.body.appendChild(heading);
    }

Github is used to store code. We use git on our computers to keep our computers in sync with the code stored in Github. Github is used very widely in industry and we will also use it for your professional portfolio, to show to potential employers as you grow your skillset in web development. GitHub is essentially a big database of source code for people all around the world. You can make your source code public or private and configure many different permissions and settings. Each project has its own repository, or "repo". This repository will be a clone of the one on your local computer. Thus, you will have two copies of your code, one on your local computer and one at GitHub.

Among others, you should be familiar with the following git commands:

  • git clone

  • git remote

  • git status

  • git add .

  • git commit -m ""

  • git push
