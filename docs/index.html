Course Description

The course content found in these webpages was developed to fulfill the Performance Based Objectives of the Web Design with HTML & CSS Beginners Class course developed and managed by the Digital Workshop Center educational institution.

Performance Based Objectives

Upon successful completion of the Web Design with HTML & CSS Beginners Classes, students will be able to:

  • Understand basic HTML and CSS basic terminology

  • Create a functional web site using HTML5 and CSS3

  • Structure content and apply styles to your page

  • Add hyper-links

  • Develop an advanced webpage

  • Create intriguing, modern layouts

  • Use advanced CSS techniques including layout and positioning

  • Incorporate Metadata and Multimedia

  • Validate your code

  • Learn the latest CSS3 features including shadows, gradients

  • Optimize a website for accessibility & search engines

  • Learn how to use the browser inspect tools

Course Pre-requisites

Coming into this course

  • Basic digital literacy including file management is required.

  • The DWC course Introduction to Design Fundamentals is recommended, but not required.

Course Lessons

During the three-week duration of this course, the class meets a total of six times. The lesson topics are as follows:

The best way to learn HTML and CSS (and any other programming language) is to frequently apply what you are learning.

  • There will be class activities throughout the course, but you will notice that the bulk of new content is taught during lessons 1, 3 and 5. During these class sessions, you will be exposed to a lot of new content. This will prepare you for the next two assignments as well as the class work day that will follow.

  • The other lessons will still present some new material, however the bulk of Lessons 2, 4 and 6 will be lab time: an opportunity for you to write code, applying what you've learned, with help at hand for any questions or issues that may arise.

  • As a disclaimer, be aware that you will be asked to share your screen most of the class sessions. Each lesson will have a homework assignment that is to be completed before the next class. Class will always commence with presentations. This isn't just an opportunity to present your code and your work, but it is also an opportunity to get specific help to unique questions you may have. Students that come with questions often feel that this is one of the most beneficial parts of the lesson.
