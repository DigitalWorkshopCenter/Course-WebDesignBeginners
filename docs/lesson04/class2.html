Hands-on Exercise 2
More Tests

Overview

Now that you have written some tests, we are going to get more experience working with tests, async, props, mock data, and more.

Instructions

  • Download This code

  • Run the following commands:

    • npm install

    • npm install

    • npm test

Look through this project and investigate the code. Do your best to answer the following questions:

  • What do you notice about the tests that are being run?

  • How is the data getting into the tests?

  • How is mock data being utilized?

  • What is the purpose of mock data?

  • Where can you find the official documentation for Jest?

  • How does create-react-app help us get started with testing?

  • Where can I find the documentation on the React-Testing-Library?
