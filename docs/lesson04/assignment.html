Assignment #4

Overview

For your assignment, you will gain experience working with CSS Media Queries. This will be very important for your lab project as well as any work that you ever do with HTML and CSS. The goal of this exercise is to use CSS media queries to make a webpage appear different for printing and for different window sizes.

Instructions

  1. Download this zip file, which already has HTML and CSS for a header, navigation, content, sidebar, and footer. This is what it looks like:

  2. Add a CSS media query and appropriate styles so that the webpage looks like the screenshot below when printed. Specifically:

    • The header, navigation, and footer should be hidden.

    • The sidebar should display under the main content.

  3. Add a CSS media query and appropriate styles so that the webpage looks like the screenshots below when resized to smaller widths. Specifically:

    • The sidebar should be hidden.

    • The body should have no padding.

    • The images shouldn't exceed the width of the window.

    • The navigation items should each be on their own line.

    • The header should be fixed, so that it stays at the top after scrolling.

  4. 240x320: small phone

  5. 320x470 Android normal

Stretch Challenges

  • Think of another way to make these views look better. Is there anything that you see that isn't necessarily ideal on any potential size of this webpage? Try to make the needed changes to make it look the way you want it to look.
