Hands-on Exercise 1
Rocking the Developer Tools

Overview

Get comfortable using the developer tools by using them on this webpage to change a bunch of stuff.

Instructions

  • Open the developer tools on this page. Notice the three vertical dots in the top right corner. If you click there, you can change where the tools show up for you.

    Also notice the following icon...It is extremely helpful.

  • Examine the code using the Elements tab. Be sure to use the dropdown arrows to see each piece of the code.

  • Find one of the steps in these instructions, and delete it. (Select it in the Elements tab, then hit your delete or backspace key).

  • Refresh the page, and notice that you still have that step there even though you deleted it. NOTE: Any changes you make in these tools are only stored temporarily. As soon as the page is refreshed, those changes will be gone forever.

  • Drag the right side of the developer tools open more so that you can see each style being applied to the page.

  • Find one of the images in the Elements tab and modify it in the styles tab so the width is only 100px.

  • Find some of the pink text on this page. Change it to be a color of your choice.

Conclusion

Congratulations! You've learned how to use some of your developer tools. Be sure to always have these tools on hand when writing and running code because they will help you be a more efficient and proficient web developer.
