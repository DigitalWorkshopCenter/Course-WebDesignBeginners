Elements

For HTML and CSS, the Elements Tab is EVERYTHING!!! Not only can I see exactly how all of my html and css code has been rendered, but I can also see what css styles have been overridden, I can change styles and add elements to see how things will change if I modify the css or html. This can be extremely helpful if used correctly. NOTE: changes made in the browser will not be saved. You can make these changes to help you quickly identify what changes need to be made. You will then have to go to VS Code, and make these changes and save the file(s).

Console

Without JavaScript, the console will only do so much for you. But it will still show errors if you have a file that cannot be found. For example, if I try to reference an image or a css file that doesn't exist, or is in a different place, I will see those errors in the console. This is extremely helpful because then I know exactly where I need to look to fix my code.
