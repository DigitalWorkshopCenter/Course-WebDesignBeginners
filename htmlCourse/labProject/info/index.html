Website Info

This website was created for a class assignment. The class was an intro to web development class which taught HTML and CSS over the span of a few weeks. Each page was designed to give the students experience working with various aspects of html and css. These were the requirements:

Home Page

  • Must be named index.html and

  • Include a picture of you, your family, or something that represents you.

  • Includes a description about yourself.

  • Includes links to the other sections of your website within the homepage (not just the navbar)

Places Page

  • This can be either a Favorite Places or Favorite Foods page.

  • Include at least 5 favorites. Each should have a title, image and description.

Fun Times Page

  • This can be either a Fun Experiences or Funny Stories page.

  • Include at least 5 favorites. Each should have a title, an approximate date, 1 image or 1 video and a description.

Hobbies Page

  • This can be either a Hobbies or Skills page.

  • Include at least 5 items. Each should have a title, an image, and an HTML list showing how you came to have that skill or hobby.

Family Page

  • This should be a webpage of your choosing. It should fit well with the other pages in being about you.

  • You may include whatever you want to on this page, but it needs to include at least 3 images, and an HTML table.

All Pages should:

  • ...be in its own folder that includes the relevant css file and any relevant images or other media content.

  • ...have a css file for that page.

  • ...have a professional layout on desktop, tablet and mobile screens by using CSS media queries

  • ...have a Navbar that looks the same (except an active identifier to show the user which page they're on).

  • ...have valid links and valid html.
